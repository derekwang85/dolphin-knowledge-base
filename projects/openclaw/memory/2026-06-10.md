### 经验记录

- **@Value 默认值兜底**: 所有 `@Value("${...}")` 必须加 `:defaultValue`，已写入 G9 门禁
- **YAML 特殊字符**: `&` 在 YAML 中会被解析为锚点，URL 须用双引号包裹
- **mato cmd 执行**: mato 的 `cmds` 只是元数据，不会自动执行，需用户手动 Enter
- **API Key 打码**: shell 自动替换 token 字符，用 Python 子进程或文件读取绕过
- **Figma 限流**: Starter 计划 PAT 限流 399773 秒 (4.6天)，Figma API 批量调用需间隔
